```

Header: Authorization: Bearer <access_token_from_login>
Body (form-data):
  file        → select any image file
  file_hash   → SHA-256 of that file (use online tool for testing)
  captured_at → 2025-01-28T10:30:00+06:00
  evidence_type → photo
  harm_type   → harassment
  file_mime_type → image/jpeg
```
